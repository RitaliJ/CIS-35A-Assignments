Extract BankSavings to a local folder.
To compile the Bank Savings program run command from the src folder:

javac cis35/assignment4/*.java

To Run part 1 of the assignment use the command: 

java -cp . cis35.assignment4.Driver1

On execution the program will create two saver accounts and the annual interest rate will be used to calculate the monthly interest rate and update the balance.


To Run part 2 of the assignment use the command: 

java -cp . cis35.assignment4.Driver2

On execution the program will create two SpecialSavings accounts and the annual interest rate will be used to calculate the monthly interest rate and update the balance. Some amounts will also be deposited and withdrawn.
