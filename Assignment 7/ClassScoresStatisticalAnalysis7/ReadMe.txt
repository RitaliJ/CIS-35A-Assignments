Extract ClassScoresStatisticalAnalysis to a local folder.
To compile the Class statistics program run command from the root folder:

javac src/**/*.java

____

To Run the program use the command: 

java -cp src driver.Driver


On execution the program will ask call the StudentAPI implementation and read the file to populate the students. The program will then ask the user for their student ID. The program will also ask for what information the user would like to view. Based on the input, the program will print the student's scores, class statistics, or both. It will include information for both subjects.
