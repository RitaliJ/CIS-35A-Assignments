Extract SalesData to a local folder.
To compile the Sales Data program run command from the root folder:

To Run the program use the command: 

java -cp . cis35.assignment3.Driver

On execution the program will prompt for a store number for which you want to view the data. If the store input is valid, the program will prompt for the type of information you would like to view. The program will display the requested results. This process will continue to repeat. The program may be terminated at any time by the user - simply type # and press enter.
